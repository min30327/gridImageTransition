options

default options 
element : "li",
speed   : 900,
easeIn  : "swing",
easeOut  : "swing",
duration: 1000,
maxImage: 16,
animate: "random",
postion: "random" ,
loading: "img/loading.gif"
Option name Description
element Float element. 画像をネストする要素を指定します。
speed Animation speed.　アニメーションスピードを指定します。
easeIn & easeOut Easing animation.　イージングを指定します。
duration Interval speed. アニメーション間隔を指定します。
maxImage The max of the picture which carries out animation. アニメーションさせる画像の最大値。
animate The type of animation has random fade slide matrix turned. アニメーションの種類はrandom fade slide matrix turnedです。
postion It can specify, only when animate is slide. left right top bottom leftTop rightTop leftBottom rightBottom
アニメーションのを「slide」にした時のみ指定できます。left right top bottom leftTop rightTop leftBottom rightBottom
loading The path of a loading picture is specified.　ローディング画像のパスを指定します。
